# LGTM画像投稿サイトのアイデア

## 背景

チーム開発で、コードレビュー時のApprove時に`イケてる`LGTM画像を貼っておきたい
チームの雰囲気を和ませたい。いつでも使える、lgtmoonのようなサイトがあるといい。


## やりたい事
- LGTM画像がサイト上にリストされており、誰でもコピーボタンを押せば投稿用マークダウンをコピーできる
- ユーザはその投稿用マークダウンでApprove時にLGTM画像を貼り付けられる
- Github認証でユーザ登録するとユーザ専用機能が使える
- ユーザ専用機能として、画像登録機能と画像削除機能が使える
- 画像削除機能は自身の登録した画像のみ削除可能
- ユーザ専用機能として、お気に入りへの追加機能も利用可能
- 管理者機能としてユーザのアカウントを停止する機能、もしくは画像を削除する機能もつける
- 参考サイトは以下
  - https://lgtmoon.com/
  - https://lgtmoon.herokuapp.com/
- 可能な限り無料で運用したい
  - Vercelにデプロイ
  - DBはPlanetScaleを利用
