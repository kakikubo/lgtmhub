# 要求定義: 画像保存サイズを長辺400pxに縮小

## 背景
- 現状、アップロード画像は長辺800pxで保存されている
- GitHubのissue/PRにマークダウンで貼ると大きすぎる

## やること
1. `MAX_LONG_SIDE = 800` → `400` に変更
2. 詳細ページのコンテナを `max-w-[400px]` に揃える
3. sizes属性を新コンテナサイズに合わせて更新
4. ユニットテストの期待値を更新

## やらないこと
- 既存画像のマイグレーション（新規アップロードのみ適用）
- LGTM テキストオーバーレイの式の変更
- sharp + WebP (quality 85) の設定変更
